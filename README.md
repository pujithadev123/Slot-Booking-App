# Slot Booking App

A simple web application that allows users to view and book available time slots.

## Features
- View available slots
- Book a slot
- See booked slots
- Store booked slots in LocalStorage

## Technologies Used
- HTML
- CSS
- JavaScript
- LocalStorage for storing booked slots
